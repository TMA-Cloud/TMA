# Manage Folders

Organize your files with folders in TMA Cloud.

## Creating Folders

### Basic Creation

1. Navigate to where you want to create a folder
2. Click **"New Folder"** button
3. Enter folder name
4. Press Enter or click Create

### Nested Folders

- Create folders inside other folders
- Unlimited nesting depth
- Hierarchical organization

## Folder Operations

### Moving Folders

- **Drag and Drop:** Drag folder to new location
- **Move Option:** Right-click → Move → Select destination
- **Bulk Move:** Select multiple folders and move together

### Copying Folders

- **Copy Option:** Right-click → Copy → Select destination
- **Bulk Copy:** Select multiple folders and copy together
- **Recursive:** All contents copied automatically

### Renaming Folders

- **Rename Option:** Right-click → Rename
- **Inline Edit:** Click folder name to edit
- **Validation:** Prevents invalid names

### Deleting Folders

- **Delete Option:** Right-click → Delete
- **Soft Delete:\*\*** Moves to trash (not permanent)
- **Recursive:** All contents moved to trash

## Folder Organization

### Best Practices

- Use descriptive folder names
- Organize by project or category
- Keep folder structure shallow when possible
- Use consistent naming conventions

### Navigation

- **Breadcrumbs:** Navigate up folder hierarchy
- **Back/Forward:** Back and Forward buttons in the file manager header move through folder history. Mouse back/forward (side buttons) do the same when on file manager. See [Desktop App — Folder navigation](/getting-started/desktop-app#folder-navigation-backforward).
- **Search:** Find folders by name

## Folder Sharing

### Sharing Folders

- Share entire folders with share links
- All folder contents accessible via share
- New files automatically included
