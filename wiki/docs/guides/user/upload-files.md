# Upload Files

Learn how to upload and manage files in TMA Cloud.

## Uploading Files

### Basic Upload

1. Navigate to the folder where you want to upload
2. Click the **"Upload"** button (or drag and drop)
3. Select files from your computer
4. Files will upload automatically

### Drag and Drop

- Drag files directly into the file manager
- Drop them in the desired folder
- Upload progress shown in real-time

### Upload Limits

- File size limited by your storage quota
- Upload rate limited to 50 uploads per hour
- Storage limit enforced at upload time

## File Management

### Viewing Files

- **Grid View:** Thumbnail view of files
- **List View:** Detailed list with metadata
- **Sort Options:** Name, date, size, type

### File Operations

- **Download:** Click to download files
- **Rename:** Right-click → Rename
- **Move:** Drag and drop or use Move option
- **Copy:** Right-click → Copy
- **Delete:** Right-click → Delete (moves to trash)
- **Star:** Mark files as favorites

## File Types

### Supported Files

- All file types supported
- MIME type detection automatic
- Preview for images and documents

### Document Editing

- **OnlyOffice Integration:** Edit .docx, .xlsx, .pptx files
- **PDF Viewing:** View PDF files in browser
- **Image Viewing:** View images with zoom

## Best Practices

- Organize files in folders
- Use descriptive file names
- Star important files for quick access
- Regularly clean up trash
