# Trash & Restore

Recover deleted files and manage your trash in TMA Cloud.

## Trash System

### Soft Delete

- Files moved to trash (not permanently deleted)
- Restorable within retention period
- Automatic cleanup after 15 days

## Accessing Trash

### Trash Section

1. Navigate to **"Trash"** in sidebar
2. View all deleted files
3. See deletion dates

### Trash Contents

- Files and folders
- Original location information
- Deletion timestamps
- Size information

## Restoring Files

### Single File Restore

1. Right-click on file in trash
2. Select **"Restore"**
3. File restored to original location

### Bulk Restore

- Select multiple files
- Click **"Restore"**
- All selected files restored

### Restore Location

- Files restored to original folder
- If original folder deleted, restored to root
- Path information preserved

## Permanent Deletion

### Delete Permanently

1. Right-click on file in trash
2. Select **"Delete Permanently"**
3. File permanently removed (cannot be recovered)

### Automatic Cleanup

- Files automatically deleted after 15 days
- Background worker handles cleanup
- No manual intervention needed

### Bulk Permanent Delete

- Select multiple files
- Click **"Delete Permanently"**
- All selected files permanently removed

## Trash Management

### Best Practices

- Regularly review trash
- Restore accidentally deleted files
- Permanently delete sensitive files
- Let automatic cleanup handle old files

### Storage Impact

- Trash counts toward storage quota
- Permanently delete to free space
- Automatic cleanup frees space
